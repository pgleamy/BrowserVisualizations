ASTEROIDS Clone

Not to bad of a clone of Asteroids I made. Was one of my fav games as a kid. Use the up, left, and right arrows to navigate, and spacebar to fire. Doesn't work on a cell phone. Need a computer.

Force Directed Graph

Each node is connected to the others by stretchy bonds, but each node also repels the other nodes. So the system seeks equilibrium. Zoom and pan with mouse, drag nodes around to see the effects. Doesn't work on cell. Need a computer.

Force Directed Graph

Pi Visualization

Displays Pi for up to 1 trillion digits as a continous grayscale heat-map on a 100 x 35 marching grid. Requires Internet for API calls to Google to get the Pi digits.

Pi Visualization
