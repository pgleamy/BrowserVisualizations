Visualizations & Games

A collection of JavaScript artistic visualizations and games I've made. Each visualization is in a single .html file. Most are interactive in some way. Browser full screen mode is better. My favorite visualization is the Solar System. The games are simple and usually just use the arrow keys, spacebar etc. All of these are created from my imagination in collaboration with artificial intelligence coding tools.

Solar System 3D Visualization (left mouse rotate / right mouse translate / scroll wheel zoom in/out)

Atoms Visualization (mouse drag)

Fibonaacci Mesh Visualization (not interactive)

FlabbyTron Game (down-arrow to start/re-start game and to play)

Wind Flyer Game (up/down arrow keys to avoid obstacles)
